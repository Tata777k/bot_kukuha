# @Currensyconverter_bot

# Телеграм бот для конвертации валют. Имя: @Currensyconverter_bot
Этот телеграм-бот знает несколько команд:

/start выдает стартовое сообщение и пишет, какие команды еще он умеет выполнять;
/help напоминает, какие команды умеет выполнять бот;
/values выписывает все валюты, которые знает бот;
<валюта 1> <валюта 2> <количество> выводит значение количества валюты 2 в валюте 1.
